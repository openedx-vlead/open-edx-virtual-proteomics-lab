1. Receptor building – The receptor is downloaded from RCSB PDB



 Image 1: PDB page displaying the protein of interest



  1. After downloading the pdb format of the protein, remove the water molecules by editing the TEXT of the protein.





  1. Optimize the protein using the parameters available in the software. Upload the pdb file and select the GA parameter likes Population size, Generations etc which are necessary for docking accuracy. Prepare the binding site using the same panel displayed. Set the output path to store the prepared structure.
  1. The active site is predicted and later the ligand is uploaded. Arrange all the parameters such as number of pose to be obtained and score and begin docking.





  1. The score is calculated and the fitness is displayed. The data is analyzed and the most optimum ligand in the pose with the least energy is selected.





  1. The complex can be viewed and checked for the orientation of the ligand with the receptor. We can also visualize the interacting amino acids; the hydrogen bond formation can also be seen. Further amino acids forming hydrophobic interactions are also shown.

LigPlot Analysis - After the docking results are obtained, the interaction can be studied using program like LigPlot that automatically plot ligand-protein interactions. Here, the ligand bound protein is used as the input file to generate the 2D plot of the hydrogen bond interaction and also the hydrophobic interaction.
