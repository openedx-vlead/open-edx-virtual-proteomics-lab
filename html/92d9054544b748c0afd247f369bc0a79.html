Identification of similar regions in protein sequences to determine the functional, structural and evolutionary relationships between them.

Step by step Procedure:

  1. Choose the type of alignment to be performed whether pairwise or multiple sequence.
  1. Input the subject protein sequence of your choice or upload it through the browse tab in the FASTA format.
  1. Input the query protein sequence or upload it using the browse tab in FASTA format.
  1. Select the general parameters like Max target sequences to align maximum length of a sequence in a row, word size, expected threshold and query matches for pairwise alignment. Select protein weight matrix, Clustering method and file formats for multiple sequence alignment. Gap opening and extension should be kept constant.
  1. Select the scoring parameters. Select the matrix to assign scores for aligning possible pair of residues. Select BLOSUM or PAM for stronger and weaker alignments or distantly related sequences respectively.
  1. Select gap costs so as to decrease the number of gaps in the alignment.
  1. By default, "low complexity region" is selected to filter out the repeated region and improve the sensitivity of sequence similarity search.
  1. Hit the align tab to see the result.
